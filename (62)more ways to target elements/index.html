62- THE DOM: More ways to target elements.

firstChild:
.firstChild returns the first child node. An element node, a text node, or a comment node
lastChild:
.lastChild returns the last child node. An element node, a text node, or a comment node.
nextSibling:
.nextSibling returns the next sibling node: An element node, a text node, or a comment node.
previousSibling:
.previousSibling returns the previous sibling node. An element node, a text node, or a comment node.
firstElementChild:
.firstElementChild returns the first child element (ignores text and comment nodes).
lastElementChild
.lastElementChild property returns the last child element (ignores text and comment nodes).
nextElementSibling:
nextElementSibling returns the next sibling element (ignores text and comments).
previousElementSibling:
.previousElementSibling returns the previous sibling element (ignores text and comments).
