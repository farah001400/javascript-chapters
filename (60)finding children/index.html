60- THE DOM: Finding Children

parentNode:
.parentNode returns parent node of an element.
children:
.children returns a live HTMLCollection of all child elements. .children returns only element nodes like h1 to h6, p, span etc
childNodes:
.childNodes returns a live NodeList of all child nodes, including text nodes, comments, etc.
